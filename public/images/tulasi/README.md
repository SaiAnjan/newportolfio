Please add the following images extracted from your project thesis PDF:

1. **conversational-workflow-diagram.png** (Page 10)

2. **saas-dashboard-wireframe.png** (Page 15)

5. **conversational-design-principles.png** (Page 25)
   - Conversational design principles diagram or summary visual

## Image Requirements

- **Format:** PNG or JPEG
- **Resolution:** 1920px width minimum (for high-quality display)
- **File Size:** Under 500KB each (optimize for web)
- **Naming:** Use exact filenames listed above

## How to Extract

1. Open your PDF thesis in Preview (Mac) or Adobe Acrobat
2. Navigate to the specified page
3. Select and crop the relevant section
4. Export as PNG at 300 DPI
5. Save with the exact filename in this directory

Once images are added, they will automatically appear on the Tulasi project page.
