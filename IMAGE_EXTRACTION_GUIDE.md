## How to Extract Images from PDF

### Option 1: Using Adobe Acrobat
1. Open the PDF in Adobe Acrobat
2. Go to Tools > Export PDF
3. Select "Image" > "PNG" or "JPEG"
4. Choose the specific pages (10, 15, 18, 22, 25)
5. Export and crop as needed

### Option 2: Using Preview (Mac)
1. Open the PDF in Preview
2. Select the page you want
3. Go to File > Export
4. Choose PNG or JPEG format
5. Set resolution to 300 DPI or higher
6. Crop the specific section you need

### Option 3: Using Online Tools
- Use tools like PDF24, SmallPDF, or ILovePDF to extract pages
- Then use image editing software to crop specific sections

### Option 4: Screenshot Method
1. Open PDF at 200% zoom for better quality
2. Take screenshots of the specific sections
3. Use image editing software to clean up and optimize

## Image Optimization

After extracting, optimize images for web:
- **Resolution:** 1920px width minimum for high-quality display
- **Format:** PNG for diagrams with text, JPEG for photos
- **File Size:** Compress to under 500KB each if possible
- **Tools:** Use TinyPNG, ImageOptim, or similar tools
